Belajar CSS

Apa itu CSS

CSS adalah singkatan dari Cascading Style Sheets. Ini adalah bahasa gaya yang digunakan dalam pengembangan web untuk menggambarkan presentasi dan gaya dari dokumen yang ditulis dalam HTML (Hypertext Markup Language). CSS digunakan untuk mengendalikan bagaimana elemen-elemen pada halaman web ditampilkan, termasuk tata letak, warna, jenis huruf, jarak, dan lainnya.

Bagaimana menggunakan CSS

Ada 3 cara menggunakan CSS :

  1. Inline Styles : menuliskan code css langsung pada tag yang akan diubah atau diatur style nya
  2. Internal/Embedded Styles: menuliskan kode CSS pada bagian header dokumen HTML
  3. Eksternal/Linked Styles: Menuliskan kode CSS pada file terpisah dan dipanggil tag link dari dokumen HTML
